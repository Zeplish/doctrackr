Update DocTrackr seed data and reminder email behavior.

==================================================
1. DOCUMENT TYPE MODEL UPDATE
==================================================

Document types should support an optional external form/download URL.

Add field to DocumentType:

formUrl String? / nullable

Purpose:
For student compliance forms, parents should receive direct download links in reminder emails.

Do NOT store completed documents.

This is only for blank form download links.

==================================================
4. EMAIL TEMPLATE UPDATE FOR PARENTS
==================================================

For STUDENT reminder emails only:

If the document type has formUrl, include a clear download section in the HTML email.

Example section:

"Download Required Form"

Button:
Download [Document Name]

Button link:
[formUrl]

Supporting text:
Please download, print, complete, and return this form to the daycare.

This should appear prominently in the email so parents do not need to ask the admin team for the form.

Also include the same URL in the plain text fallback email.

==================================================
5. EMAIL TEMPLATE UPDATE FOR EMPLOYEES
==================================================

Employee reminder emails do not need formUrl buttons unless a formUrl exists.

If formUrl exists for an employee document type in the future, include the link.

==================================================
6. ADMIN UI UPDATE
==================================================

In Document Types admin screen, add/edit field:

Form Download URL

Rules:
- Optional
- Validate URL format
- Show only if provided
- Display in document type details/table as a clickable link

==================================================
7. IMPORTANT
==================================================

Do not upload or store completed documents.

Do not require file upload for student forms.

Use external formUrl links in emails.

Keep the system HIPAA-conscious and compliance-focused.
